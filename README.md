# mcbridge
Mineflayer bot that forwards messages from and to Discord

## Configuration
```json
{
    "token": "DISCORD_TOKEN_HERE",
    "guild": "GUILD_ID_HERE",
    "role": "ADMIN_ROLE_ID_HERE",
    "channel": "CHANNEL_ID_HERE",
    "host": "SERVER_IP_HERE",
    "port": 25565,
    "username": "ServerOverflow",
    "modded": false
}
```
